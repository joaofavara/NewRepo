# NewRepo

## Project setup
### Create the mongodb local
`docker-compose up`

### Run the script, creating an user on the mongo
 - To go database/script/
 - Run the setup.js `node setup.js`
  
### Start the application

- Front-end:
  - `npm run serve`
 
- Back-end
  - Inside the server folder
  - `nodemon index.js`
